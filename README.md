# 🛒 Sellora — Marketplace of Social Commerce

**Sellora** — це сучасний маркетплейс, що трансформує класичний онлайн-шопінг у соціальну взаємодію.
Ми допомагаємо малому бізнесу залучати клієнтів через вірусні механіки, а покупцям — отримувати
найкращі ціни.

---

## 🚀 Кілер-фіча: "Купуй разом" (Group Buy)

Наша головна перевага — механізм колективних знижок. Це дозволяє знизити вартість товару завдяки
об'єднанню зусиль покупців.

### Як це працює?

1. **Вибір товару:** Покупець бачить дві ціни: стандартну (наприклад, 100 грн) та ціну для "Групової
   покупки" (наприклад, 90 грн).
2. **Створення групи:** Покупець ініціює групову покупку, стаючи лідером групи.
3. **Залучення друзів:** На товар встановлюється таймер (наприклад, 24 години) та необхідна
   кількість учасників (наприклад, 3 особи).
4. **Успіх:** Якщо протягом визначеного часу група заповнюється, всі учасники отримують товар за
   акційною ціною.
5. **Ризики:** Якщо група не назбиралася, замовлення скасовується (кошти повертаються), або покупець
   може доплатити до повної вартості.

**Вигода:** Продавець отримує 3 продажі замість 1 без витрат на маркетинг, а покупці економлять
кошти.
