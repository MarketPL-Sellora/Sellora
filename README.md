## 📖 About

**Sellora** — це сучасний маркетплейс, що трансформує класичний онлайн-шопінг у соціальну взаємодію. Ми допомагаємо малому бізнесу залучати клієнтів через вірусні механіки, а покупцям — отримувати найкращі ціни.

> **💡 Our Mission:** Bridge the gap between social engagement and e-commerce — empowering small businesses with zero-cost marketing through the power of community buying.

---

## 🚀 Killer Feature: Group Buy

> _"Купуй разом"_ — Our signature collective discount mechanism that reduces prices through the power of community.

Наша головна перевага — механізм колективних знижок. Це дозволяє знизити вартість товару завдяки об'єднанню зусиль покупців.

### ⚙️ Як це працює?

| Крок | Дія | Деталі |
|:---:|:---|:---|
| **1** | 🏷️ **Вибір товару** | Покупець бачить дві ціни: стандартну (напр. `100 грн`) та ціну для групової покупки (напр. `90 грн`). |
| **2** | 👑 **Створення групи** | Покупець ініціює групову покупку, стаючи **лідером групи**. |
| **3** | 📢 **Залучення друзів** | На товар встановлюється таймер (напр. `24 години`) та необхідна кількість учасників (напр. `3 особи`). |
| **4** | ✅ **Успіх** | Якщо група заповнюється вчасно — всі учасники отримують товар за **акційною ціною**. |
| **5** | ⚠️ **Ризики** | Якщо група не назбиралася — замовлення скасовується (кошти повертаються), або покупець може доплатити до повної вартості. |

## 🏗️ Tech Stack

| Layer | Technology | Description |
|:---|:---|:---|
| **🎨 Frontend** | Vue.js 3 + Tailwind CSS | Reactive SPA with utility-first styling |
| **⚙️ Backend** | Java Spring Boot 3 | RESTful API & business logic |
| **🗄️ Database** | PostgreSQL 16 | Relational data with ACID compliance |
| **🐳 DevOps** | Docker Compose | Containerized multi-service orchestration |
| **📄 API Docs** | OpenAPI / Swagger | Interactive API documentation |
