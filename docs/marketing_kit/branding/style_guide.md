## 🏷 Ідентичність бренду

### Назва

> **Sellora** — маркетплейс нового покоління з груповими знижками та гарантією якості.

Назва бренду завжди пишеться з великої літери **Sellora** (ніколи `SELLORA`, `sellora` або `SeLLora`).

### Слоган

> *«Transforming classic online shopping into a viral social experience.»*

Слоган використовується англійською мовою як частина глобальної айдентики бренду. При потребі може супроводжуватись україномовним варіантом:

> *«Маркетплейс нового покоління з груповими знижками та гарантією якості.»*

### Логотип

| Варіант | Файл | Опис |
|:--------|:------|:-----|
| Основний логотип | `logo_primary.png` | Повнокольоровий логотип на темному тлі |
| Іконка (фавікон) | `favicon.svg` | Спрощена іконка для вкладок браузера |

**Правила використання логотипу:**

- Мінімальний відступ навколо логотипу — не менше висоти літери «S» у логотипі
- Логотип завжди супроводжується текстовим написом **Sellora** шрифтом `Unbounded Bold`
- Кольори напису логотипу: `orange-500` (`#f97316`) на темному фоні
- Логотип має фіксований розмір `32×32 px` (`w-8 h-8`) із заокругленням `rounded-lg` (8 px)

## ✏️ Типографіка

### Сімейства шрифтів

Проєкт Sellora використовує два основних веб-шрифти, що визначають характер бренду:

| Шрифт | Роль | Стиль | Використання |
|:------|:-----|:------|:-------------|
| **Unbounded** | Display / Заголовки | `font-['Unbounded']` | Логотип, заголовки сторінок (`h1`, `h2`), ціни товарів, назви секцій категорій, мітки кабінету, CTA-кнопки, статистика Hero-банера, таймер відліку групових покупок |
| **Onest** | Body / Основний текст | `font-['Onest']` | Базовий шрифт всього додатку (`font-['Onest']` на `<body>`). Навігація, описи товарів, поля форм, мітки полів, посилання футера, кнопки фільтрів, модальні вікна, тексти повідомлень |
| **Segoe UI Symbol** | Допоміжний | `font-['Segoe_UI_Symbol']` | Зірки рейтингу (`★☆`) — системний шрифт для символів Unicode |

### Ваги шрифтів (Font Weight)

| Вага | Клас Tailwind | Використання |
|:-----|:-------------|:-------------|
| **Normal (400)** | `font-normal` | Основний текст, описи, мітки полів, посилання |
| **Medium (500)** | `font-medium` | Назви категорій, текст навігаційних кнопок |
| **Semibold (600)** | `font-semibold` | Кнопки CTA, активні мітки, підзаголовки |
| **Bold (700)** | `font-bold` | Логотип, ціни, заголовки, бейджі |
| **Extrabold (800)** | `font-extrabold` | Великі суми (підсумок замовлення) |
| **Black (900)** | `font-black` | Заголовки Hero-банера, мітки секцій кабінету, сторінка 404 |

### Розміри тексту

| Розмір | Клас Tailwind | Застосування |
|:-------|:-------------|:-------------|
| `9px` | `text-[9px]` | Бейджі картки товару, лічильник кошика |
| `10px` | `text-[10px]` | Бейджі статусу, промо-мітки, бейджі HeroBanner |
| `11px` | `text-[11px]` | Підписи до статистики |
| `12px` | `text-xs` | Навігація, мітки, фільтри, мета-інформація, футер |
| `14px` | `text-sm` | Основний текст, описи, поля форм, кнопки |
| `16px` | `text-base` | Базовий розмір, текст CTA великих кнопок |

| `20px` | `text-xl` | Ціна на картці товару, заголовки кабінету |

### Заокруглення кутів (Border Radius)

| Токен Tailwind | Значення | Призначення |
|:---------------|:---------|:------------|
| `rounded-full` | `9999px` | Аватари, точки індикаторів, лічильники, бейджі, слайдер-доти |

### Анімації та переходи

| Елемент | Тривалість | Тип | Опис |
|:--------|:-----------|:----|:-----|
| Hover кнопок | `150ms` | `transition-all` | Плавна зміна кольору та тіні |
| Hover карток | `300ms` | `ease-out` | Підйом картки (`-translate-y-1.5`) та поява тіні |
| Масштаб зображень | `700ms` | `transition-transform` | Збільшення зображення при hover (`scale-105`) |
| Slide-fade (слайдер) | `400ms` | `ease` | Перехід між слайдами Hero-банера |
| Active (натискання) | миттєво | `scale-[0.97-0.98]` | Мікро-анімація натискання |
| Focus ring | `150ms` | `transition-colors` | Поява фокус-кільця (`ring-orange-500/40`) |
| Backdrop blur | — | `backdrop-blur-sm` / `backdrop-blur-[10px]` | Ефект розмиття фону для хедера та модальних оверлеїв |
| Авто-слайдер | `5000ms` | `setInterval` | Автоматична зміна слайдів Hero-банера |

### Стани елементів

| Стан | Візуальний ефект |
|:-----|:----------------|
| **Default** | Стандартний вигляд компонента |
| **Hover** | Освітлення фону, підсилення тіні, зміна кольору тексту, підйом картки |
| **Active / Pressed** | Зменшення масштабу на 2–5% (`scale-[0.95-0.98]`) |
| **Focus** | Кільце фокусу `ring-2 ring-orange-500/50` або `outline-orange-500/40` |
| **Disabled** | Прозорість `opacity-50-70`, `cursor-not-allowed`, блокування hover/active |
| **Loading** | Текст «Завантаження...» / «Додаємо...», `disabled` стан кнопки |

### Принципи дизайну

1. **Dark-First** — увесь інтерфейс побудований на темній палітрі з акцентом на помаранчевих тонах
2. **Glassmorphism** — використання `backdrop-blur` та напівпрозорих фонів для хедера та оверлеїв
3. **Depth through Shadows** — кастомні тіні з помаранчевим свіченням для створення ієрархії
4. **Micro-interactions** — плавні анімації натискання, hover-ефекти підйому карток, масштабування іконок
5. **Консистентність** — єдина система заокруглень (`xl` / `2xl`), відступів (кратні 4 px) та кольорів по всьому інтерфейсу
